Connects your WordPress site to MVP Affiliate.

== Description ==

MVP Affiliate manages your affiliate blog from a single dashboard:
publish AI-written reviews, push sidebars and banners, sync your bio and
social links, control footer content, and more — without touching
WordPress.

This plugin installs the bridge that lets the MVP Affiliate dashboard
talk to your site.

== Installation ==

1. Upload the plugin zip via Plugins → Add New → Upload Plugin
2. Activate
3. Visit the **MVP Affiliate** menu in your WordPress admin
4. Click **Install Kadence theme** (one click)
5. Click **Generate Connection Token** and copy the token
6. Paste the token in your MVP Affiliate dashboard
